# 🤖 AI Trader Pro — Telegram Mini App

Профессиональный торговый терминал с AI-сигналами внутри Telegram.

## ✨ Возможности

- 📊 **Живой график** (Chart.js) с анимацией
- 💱 **4 категории**: Форекс, Крипто, Сырьё, Акции
- 🎯 **Генерация сигналов** с уровнями входа, SL и TP
- 📈 **Бегущая строка** котировок
- 📋 **Копирование сигнала** в буфер обмена
- 🎨 **Премиум дизайн** с градиентами и анимациями
- 📱 **Полная адаптация** под Telegram WebApp

## 🚀 Быстрый старт

1. Скачай репозиторий
2. Залей на **GitHub** (публичный репозиторий)
3. Включи **GitHub Pages**: Settings → Pages → main → Save
4. В @BotFather: Menu Button → укажи ссылку на Pages
5. Готово! Бот открывает терминал.

## 🛠 Технологии

- Chart.js — графики
- Telegram WebApp SDK
- Vanilla JS (без фреймворков)
- CSS Custom Properties
